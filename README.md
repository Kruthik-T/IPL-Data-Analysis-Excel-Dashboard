# 🏏 IPL Data Analysis Dashboard using Excel

## 📌 Project Overview

This project focuses on analyzing Indian Premier League (IPL) data using Microsoft Excel. The objective was to transform raw IPL match and ball-by-ball data into an interactive dashboard that provides insights into team performance, tournament winners, player statistics, and overall IPL trends.

The project demonstrates data cleaning, data transformation, pivot tables, dashboard design, and data visualization skills using Excel.

---

## 🎯 Project Objectives

* Analyze IPL match and ball-by-ball data.
* Track IPL winners across seasons.
* Evaluate team performance.
* Explore player and match statistics.
* Create interactive Excel dashboards.
* Present insights through visualizations.
